### How to start the project? 
```docker-compose up```

### Where are the default enviroment variables? 
```nano .env``` 
_File .env contains the following parameters:_ 
- DB_ENGINE 
- DB_NAME 
- POSTGRES_USER 
- POSTGRES_PASSWORD 
- DB_HOST 
- DB_PORT
